# ミッション 01: 地上掃討演習

## 概要
平坦なフィールド上に配置された静止地上ターゲットを全て撃破するミッション。
プレイヤーは操作に慣れるための最初の訓練ミッション。

## マップ
- **地形**: 平坦な草原（既存の `DrawGround()` タイル地面）
- **範囲**: 特に制限なし（敵は原点付近 半径500m以内に配置）
- **天候**: 晴天（固定）

## 敵ターゲット
- **種類**: 静止地上ターゲット（動かない）
- **モデル**: 仮モデル（sphere.obj をスケール調整で代用 → 後で専用モデルに差し替え）
- **数**: 5体
- **HP**: 各50
- **配置座標**:

| No. | X | Y | Z | 備考 |
|-----|------|-----|------|------|
| 1   | 100  | 0   | 200  | 正面やや右 |
| 2   | -80  | 0   | 300  | 正面やや左 |
| 3   | 200  | 0   | 450  | 遠方右 |
| 4   | -150 | 0   | 500  | 遠方左 |
| 5   | 50   | 0   | 700  | 最遠方 |

## 自機装備
- **Gunpod** × 1
  - 弾数: 200発
  - 発射レート: 秒間10発
  - 弾丸ダメージ: 10 /発

## クリア条件
- 全5体の敵ターゲットを撃破

## 失敗条件
- 自機が地面に衝突（高度 ≤ 0）
- （自機HP = 0 は将来的に追加。初回ミッションでは敵が反撃しないため不要）

## シーンフロー
1. **TitleScene** → SPACE押下
2. **StageScene** （操作案内オーバーレイ表示 → 任意キーで閉じる）
3. 全敵撃破 → **ClearScene**
4. 地面衝突 → **ResultScene**（ミッション失敗）
5. Clear / Result → SPACE押下 → **TitleScene**
